# Campamento ChipUSM 2025: Track Digital

Este año la edicion digital del campamento ChipUSM esta orientada a **System on Chip** (SoC), los  invitamos a aprender acerca de como se diseña un circuito integrado que conlleva procesadores, interconexion y perifericos varios.


La estructura del curso se basa en 5 modulos:

- Modulo 1: Introducción e instalado de herramientas
- Modulo 2: RTL y Simulación
- Modulo 3: Síntesis
- Modulo 4: Implementación.
- Modulo 5: Implementsción del SoC Croc con Librelane.
